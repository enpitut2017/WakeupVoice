# WakeUpVoice 
自分の声で解除する目覚ましアプリ

## エレベータピッチ
普通の目覚ましでは起きれない人向けの目覚ましアプリです。事前に設定したワードを自らが大きな声で発声し，音声認識により正しいと判断されなければアラームが解除されません。朝から元気に声を出すことで覚醒を促します。
